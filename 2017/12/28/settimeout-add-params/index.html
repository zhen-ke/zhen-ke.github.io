setTimeout传参数

发表于 2017-12-28 更新于 2021-04-09 分类于 Web Developer 阅读次数：

除了前两个参数，setTimeout还允许添加更多的参数。它们将被传入推迟执行的函数（回调函数）。

使用bind方法，把多余的参数绑定在回调函数上面，生成一个新的函数输入setTimeout
